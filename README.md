# RestroBiz

**Live Demo:** https://food-restaurant-manageme-2b03a.web.app

RestroBiz হলো একটি online food gallery এবং restaurant management web application। এখানে users account create করতে পারে, food items দেখতে এবং add করতে পারে।  

---


## Features

- **User Authentication:** Register/Login using Email & Google
- **Profile Management:** Display name & profile photo
- **Food Gallery:** View and add food items
- **Order Management:** Track your own orders
- **Responsive Design:** Works on both mobile and desktop
- **Modal Forms:** Submit food items or feedback easily
- **Backend Integration:** Fetch data from server and store in MongoDB
- **Firebase Hosting:** Live deployment

---

## Tech Stack

**Frontend:**  
- React.js  
- React Router  
- Tailwind CSS  
- Lottie Animations  
- react-helmet  

**Backend:**  
- Node.js  
- Express.js  
- MongoDB  

**Authentication & Hosting:**  
- Firebase Authentication  
- Firebase Hosting
